# Systems Engineer Portfolio Platform

A production-grade developer portfolio for Barekegn Asefa, showcasing full-stack systems engineering capabilities.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animation:** Framer Motion
- **Content:** MDX
- **Testing:** Fast-check (Property-Based Testing)
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
